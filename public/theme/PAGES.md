# Static HTML Theme Pages

## Overview
This theme contains 7 complete HTML pages extracted from the Siliconile website.

**Description**: The main landing page featuring:

## Technical Features

1. **Responsive Design**: All pages work on mobile, tablet, and desktop
2. **TailwindCSS**: Utility-first CSS framework
3. **Alpine.js**: Lightweight JavaScript for mobile menu
4. **Static HTML**: No server-side processing required
5. **SEO Optimized**: Meta tags and semantic HTML
6. **Accessible**: Proper heading structure and ARIA labels
